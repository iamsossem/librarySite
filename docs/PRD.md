# [PRD] 수원시 도서관 포털: 도서 검색 UX 및 정보 접근성 개선

## 1. 프로젝트 배경
현재 수원시 도서관 포털은 통합 서비스 제공이라는 명목하에 방대한 정보를 나열하고 있으나, 실제 사용자(특히 디지털 취약계층)는 원하는 정보를 찾는 데 과도한 인지적 노력을 기울이고 있음. 특히 모바일 환경에서 도서 소장 상태를 확인하는 과정이 불투명하여 서비스 이탈률이 높고 오프라인 방문 시 헛걸음을 하는 사례가 빈번함. 이를 개선하여 **'누구나 쉽고 빠르게 책을 빌릴 수 있는 환경'**을 구축하고자 함.

---

## 2. 문제 정의 (The Problem)
* **정보 과부하:** 검색 결과 리스트에서 도서관별 상태 정보가 텍스트 위주로 빽빽하게 나열되어 시독성이 현저히 떨어짐.
* **행정 중심 용어:** '관외대출중', '상호대차' 등 일상적이지 않은 용어 사용으로 인해 직관적인 상태 파악이 어려움.
* **접근성 부재:** 노안이 있는 고령층이나 시각 약자를 위한 시각적 장치(색상, 대비, 아이콘)가 부족하여 정보 소외 현상 발생.

---

## 3. 핵심 UX 가설
> **"만약 우리가 도서 검색 결과의 '대출 가능/불가' 상태를 텍스트 대신 선명한 컬러 뱃지(초록/빨강)로 시각화한다면, 사용자들은 내가 원하는 책을 빌릴 수 있는지 판단하는 시간을 획기적으로 줄이는 긍정적인 결과를 얻을 것이다."**

---

## 4. 타겟 페르소나
* **이름:** 박순자 (68세, 전업주부)
* **특징:** 손주 육아를 돕고 있으며, 손주를 위해 도서를 대신 대출해야 하는 상황.
* **IT 숙련도:** 하 (카카오톡 사용 가능, 복잡한 표와 작은 글씨에 거부감).
* **치명적 Pain Point:** 돋보기를 써도 '대출가능'과 '대출중' 글자가 구분되지 않아 중도 포기함.

---

## 5. 핵심 요구사항 및 기능 명세

| 구분 | 요구사항 명칭 | 기능 상세 기술 | 우선순위 |
| :--- | :--- | :--- | :--- |
| **UI/UX** | **도서 상태 컬러 뱃지** | - 대출 가능: 초록색 뱃지 + 체크(V) 아이콘<br>- 대출 불가: 빨간색 뱃지 + 엑스(X) 아이콘 | 필수(P0) |
| **콘텐츠** | **친숙한 언어 순화** | - 관외대출가능 → **지금 대출 가능**<br>- 관외대출중 → **현재 대출 중** | 필수(P0) |
| **기능** | **단골 도서관 퀵 필터** | - 검색 결과 상단에 사용자가 지정한 도서관만 모아보는 스위치 버튼 제공 | 높음(P1) |
| **접근성** | **터치 영역 및 폰트 확장** | - 도서 리스트 카드 크기를 키우고 최소 폰트 크기를 16px 이상으로 유지 | 필수(P0) |

---

## 6. 성공 측정 지표 (Success Metrics)
* **도서 상세 페이지 진입 대비 대출 예약 완료율 증가:** 정확한 정보 탐색 성공 의미.
* **검색 결과 페이지 체류 시간 감소:** 정보 해석 및 의사결정 속도 향상 의미.
* **단순 사용법 관련 민원 건수 하락:** 서비스 직관성 증대 의미.
* **웹 접근성 가이드라인(KWCAG) 준수율 100% 달성.**
